# SPL Detection Queries

يحتوي هذا المجلد على ملف واحد (`detection_queries.spl`) يضم كافة استعلامات (SPL) المستخدمة في التحقيق الأمني لاكتشاف هجمات Brute-Force داخل منصة Splunk SIEM.

## محتويات الملف:
- **Brute-Force Detection:** للكشف عن المهاجمين المتكررين.
- **Top Attacking IPs:** تحديد أكثر العناوين نشاطاً في الهجوم.
- **Targeted Accounts:** تحليل الحسابات المستهدفة.
- **Timeline & Spikes:** تتبع توقيتات الهجوم ورصد الارتفاعات الحادة في المحاولات.
